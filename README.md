# Control LocalStorage — JavaScript Utility

A JavaScript utility and demo for working with the browser's localStorage API. Demonstrates CRUD operations on localStorage with a clean UI.

## Features

- Set key-value pairs in localStorage
- Get values by key
- Delete individual keys
- Clear all localStorage data
- Live display of current localStorage contents
- JSON serialization for object values

## Running

Open `index.html` in any browser — no dependencies required.

## Use Cases

- Learning the localStorage API
- Testing localStorage behavior across page reloads
- Inspecting what a web app has stored locally

## Tech Stack

Vanilla JavaScript, localStorage API, HTML5, CSS3
